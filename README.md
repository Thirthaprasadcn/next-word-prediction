# Next Word Prediction using NLP

## 📌 Project Overview

This project is a **Next Word Prediction model** built using Natural Language Processing (NLP). It predicts the most likely next word based on the input text provided by the user.

**Example:**
Input → *"I love to eat"*
Output → *"pizza"*

---

## 🚀 Features

* Predicts the next word from a given sentence
* Uses NLP and deep learning techniques
* Trained on text dataset
* Can be integrated into chatbot or typing assistant
* Simple interface (CLI or Web-based)

---

## 🛠️ Technologies Used

* Python
* TensorFlow / Keras (or PyTorch)
* NumPy
* Pandas
* NLP Techniques (Tokenization, Padding, Sequence Modeling)

---

---

## ⚙️ How It Works

1. Collect and preprocess text data
2. Convert text into sequences using tokenization
3. Train a deep learning model on sequences
4. Model learns language patterns
5. Predict the next word based on input text
